# 사이드 런지 운동 프롬프트 템플릿

## exercise_description
사이드 런지는 좌우 방향(관상면)에서의 고관절 외전/내전과 무릎 굴곡/신전의 조화를 요구하며,
중심 이동 시 지지 다리의 안정성과 움직이는 다리의 정렬이 핵심입니다.

## side_orientation
** 측면에서 바라봤을 때 사이드 런지 자세에 대한 설명
  1) 체간 정렬: 상체는 약간 앞으로 기울어지되 과도하게 숙이지 않음
  2) 지지다리 안정성: 서있는 다리는 곧게 펴서 안정성 유지
  3) 움직이는 다리 깊이: 무릎이 발끝을 넘지 않도록 하며 적절한 깊이로 내려감
  4) 뒤꿈치 접지: 움직이는 다리의 뒤꿈치가 지면에 확실히 접지

## front_orientation
** 정면에서 바라봤을 때 사이드 런지 자세에 대한 설명
  1) 무릎-발끝 정렬: 움직이는 다리의 무릎이 발끝 방향과 일직선 유지
  2) 골반 정렬: 골반이 정면을 향하며 회전하지 않음
  3) 좌우 균형: 양쪽 방향으로 대칭적인 동작 수행
  4) 발 간격: 적절한 보폭으로 좌우로 이동
  5) 상체 중심: 상체가 좌우로 과도하게 흔들리지 않음

## evaluation_criteria
** 평가 기준 (엄격 모드):
- joint_score >= 0.999 : 매우 잘함
- 0.985 ~ 0.997 : 부족
- 0.975 ~ 0.985 : 매우부족
- < 0.975 : 잘못됨

## output_guidelines
# 출력 지침:
1. {exercise}의 핵심 동작 기준과 비교해서 설명한다.
2. 주요 관절의 결과들을 보고 정확하게 피드백을 해준다. 관절 점수는 출력하지 않고 설명만 제공한다.
3. 전체적인 동작 평가 (라벨 비율 기반).
4. 상체(어깨/팔), 골반, 하체(엉덩이/무릎/발목) 강점과 부족한 점.
